Welcome to CakePhp 2 Component Mailchimp Api.

The solution from Mailchimp api, in CakePhp:

Installation

1.Add archive in components folder: app/controller/components from cakePhp. 2.Call de controller: 

var $components=array( 'Mailchimp');

3.Create function, use the component (For exemple lists)

$this->Mailchimp->MCAPI("API_KEY");
    $list = $this->Mailchimp->lists();
